# 🚀 Safe-Sub
### قالب سفارشی و حرفه‌ای برای پنل X-UI

یک اسکریپت ساده برای نصب خودکار قالب اشتراک‌گیری سفارشی روی پنل X-UI با قابلیت شخصی‌سازی کامل.

---

## ✨ امکانات

- نصب خودکار قالب در مسیر `/usr/local/x-ui/sub_templates/custom`
- پرسش ۴ سوال برای شخصی‌سازی:
  - شناسه پشتیبانی تلگرام
  - شناسه کانال تلگرام
  - شناسه ربات فروش
  - نام برند شما
- جایگزینی خودکار مقادیر در کد HTML
- قالب کاملاً ریسپانسیو و زیبا
- پشتیبانی از تم روشن/تیره
- نمایش وضعیت اشتراک، حجم مصرفی و تاریخ انقضا

---
