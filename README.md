# قالب اشتراک سفارشی SafeVPN برای 3x-ui

این پروژه به شما امکان می‌دهد قالب اشتراک (Subscription Template) پنل **3x-ui** را تنها با یک دستور نصب و شخصی‌سازی کنید.

---

## ✨ امکانات

- نصب تنها با یک دستور
- سازگار با نسخه‌های جدید 3x-ui
- دریافت خودکار آخرین قالب از GitHub
- شخصی‌سازی نام برند
- شخصی‌سازی آیدی ربات تلگرام
- شخصی‌سازی آیدی کانال تلگرام
- شخصی‌سازی آیدی پشتیبانی
- جایگزینی خودکار اطلاعات داخل قالب
- ساخت خودکار فایل‌های `index.html` و `sub.html`
- نصب سریع و بدون نیاز به ویرایش دستی فایل‌ها

---

## 📥 نصب

دستور زیر را روی سرور اجرا کنید:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/Safe-Net5/Safe-Sub/main/install.sh)
```

---

## 📝 اطلاعات موردنیاز

در هنگام نصب، اسکریپت اطلاعات زیر را از شما دریافت می‌کند:

- نام برند
- آیدی ربات تلگرام (بدون @)
- آیدی کانال تلگرام (بدون @)
- آیدی پشتیبانی (بدون @)

### نمونه

```text
نام برند:
SafeVPN

ربات:
VpnSafee_bot

کانال:
SafeVpn13

پشتیبانی:
SafeVpn5
```

---

## 📂 مسیر نصب

پس از پایان نصب، فایل‌ها در مسیر زیر قرار می‌گیرند:

```text
/usr/local/x-ui/sub_templates/safevpn
```

فایل‌های ساخته‌شده:

```text
index.html
sub.html
```

---

## ⚙️ تنظیم در پنل 3x-ui

وارد پنل **3x-ui** شوید.

از بخش **Subscription Settings** مقدار **Custom Path** را روی مسیر زیر قرار دهید:

```text
/usr/local/x-ui/sub_templates/safevpn
```

سپس تنظیمات را ذخیره کنید و در صورت نیاز سرویس را ری‌استارت کنید:

```bash
systemctl restart x-ui
```

---

## 🗑 حذف قالب

برای حذف کامل قالب:

```bash
rm -rf /usr/local/x-ui/sub_templates/safevpn
```
